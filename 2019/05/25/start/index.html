一些碎碎念

​ 故事啊要从1930年说起，据说那天北京的秋天，袁隆平出生在北京的一个普通家庭（笑）

​ 一直寻思着要搭一个小窝儿，为啥子呢，当然是吃（Ji）饱（lu）了（sheng）撑（huo）的（dian）呀（di）(￣▽￣)~*。在经过了一段时期的痛苦挣扎后，终于在被各种碾压中将这个小玩意儿搭了起来，作为第一篇博文，就姑且定为start吧。

春山空，春夜静。桂花流瓦，落花如英。

​ 春光无限好，只是枫璃秋的这个春天似乎并不那么尽如人意。大一下的枫璃秋经历了自来到这个世界的痛苦的长达数月的煎熬，因为一些很naive的事情，被人当做沙雕（笑），一段时间的失眠（以前从未有过），一段时间的痛苦反省（痛定思痛(˘•ω•˘)），虽然表面上并没有什么表现，内心却然是已经波涛汹涌，疯狂呐喊（为什么）打击!!＿|￣|○。这段时间外人自然是不知道的，内人也不一定知道（叉腰），一方面自己不怎么喜欢把自己的真实想法吐露出来，另一方面确实感觉自己太naive 了，于是就憋在了心里，埋头自闭..(｡•ˇ‸ˇ•｡)….。好在是，痛苦的东西大都已经过去了，自己终于体会到了对一个人对一些事情又爱又恨是什么感觉了（渐渐地变得异常敏感，渐渐地不再敢靠近），自己成熟了很多，正所谓“昨夜星辰昨夜风，画楼西畔桂堂东”，(,,´•ω•)ノ”(´っω•｀。)，过去了便也没有必要去刻意记住，所以就乐而忘忧吧(*´・ｖ・)。

日月有常，星辰有行。

​ 虽然把一些东西带走了，可是留下的东西却是异常珍贵的。反思之余，发现自己很平凡，发觉自己很普通是很容易的（可是却让我花了很长时间，经历了很多痛苦才明白），现实是自己和很多人没有什么区别嘛，会因为一个人一些事情影响心情很长时间，也会因为内心的失落而抑郁很久。然而发现自己还是可以努力的嘛，榆木脑袋的话就多下点功夫，内心脆弱的话就尽可能少地影响别人的生活嘛。大多数的我们都只是很平凡很平凡而已，和那些真正能够影响改变世界格局的人相比，我们实在太渺小了。可是，平凡的人生也会有平凡的快乐不是？遍历种种辛酸痛苦，要说收获最多的是什么，我想大概就是有了新的 方向，曾经很困惑的东西都不存在了，仿佛抓住了十分重要的东西，仿佛又没有，不过相比于以前的自己倒也更加能看开许多事情了。找回了一些旧友，呀不一一列举~~了，沙雕太多了不是( • ̀ω•́ )✧，一些人对于我来说是重要的，这些人不管在哪里，不管做什么，不管状态如何，只要是我认为的，对于我来说就是很重要的，毕竟我也只是一个长不大的孩纸沙雕不是？(笑)

回首向来萧瑟处，归去，也无风雨也无晴

​ 自己又摸鱼儿了那么长时间了，感觉又是什么也没干(:3」∠❀)菊花碎了一地。当然是干了一些事情的嘛，在此感谢小凌凌和聪腿的带飞以及老朱的助攻，计导大作业，一千多行代码，效果很好，小凌凌和聪腿的debug能力是真的牛批啊（偷笑），本来嘛，一周前都做的差不多了，后来老师又要改规则，这就很烦人了嘛。不过好在是在大家最后的全力以赴下第一版本效果还是很不错滴。表白小凌凌，爱了爱了♪（＾∀＾●）ﾉ，在此祭上小凌凌的一张沙雕图(｀▽´)

且将新火试新茶，诗酒趁年华

​ 绕了一大圈，以后还是要做点事情的嘛。当然是好好学习，天天向上啦。以后会用该博客记录生活，学习的一些东西，所以多多关照啦。虽然我很菜，虽然高中之前并没有接触过编程，么么么。。但是很菜的人会一点一点努力不是？(^▽^)虽然智商很一般，甚至不及大多数人，还很粗心，但是还是会一点一点地裨补缺漏不是？(oﾟ▽ﾟ)o

​ 就酱吧，有时间再更新！

​ p.s.北师是个好地方，真的是个好地方。。（偷笑）
